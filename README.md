# Sentiment Analysis

## Dependencies

- python v3.7.3 (Usually already installed on linux distros)
- pip v18.1
- newspaper3k v0.2.8
- nltk v3.4.5
- textblob v0.15.3

Installation:

```
  $ sudo apt install python3-pip -y
  $ pip3 install newspaper3k
  $ pip3 install nltk
  $ pip3 install textblob
```
